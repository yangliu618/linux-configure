### gitShell

#### 自动完成更新和上传减少用户操作 

    alias gitsh="/bin/bash /path/to/gitShell.sh"

#### 使用

    Option :    //必要参数
        ps      //执行git fetch && git rebase && git push
        fr      //执行git fetch && git rebase

    Remote :    //非必须 更新的远端（需要和Branch 同时使用)

    Branch :    //非必须 更新远端对应的分支（需要和Remote 同时使用)
